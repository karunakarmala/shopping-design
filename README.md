# AJIO Shopping App UI Design

This project is a mobile shopping app interface inspired by AJIO, designed in Figma.  
It includes 5 screens: Home, New Arrivals, Dresses, Shoes, and Accessories.

## Features
- Clean and simple mobile UI
- Product categories for easy navigation
- Attractive product display layout
- User-friendly shopping app design
- Designed in Figma for UI/UX practice

## Purpose
This project was created to improve my skills in Graphic Design and UI/UX design, especially in mobile app interface design, layout structuring, and visual presentation.
